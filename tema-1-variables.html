Tema 1 — Variables

Objetivos del tema
  • Entender qué es una variable
  • Asignar valores correctamente
  • Mostrar información por pantalla

Teoría

Una variable es un espacio en memoria donde se guarda un valor. En Python no es necesario indicar el tipo.

Ejemplo de asignación:

edad = 16
nombre = "Eva"

Código (copiable)

nombre = "Eva"
edad = 16

print("Nombre:", nombre)
print("Edad:", edad)
✅ Copiado

Salida esperada

Nombre: Eva
Edad: 16

Práctica

  1. Crea una variable llamada ciudad y muéstrala.
  2. Pide al usuario su edad y muéstrala.
  3. Reto: calcula el año de nacimiento.
